# 📤 위험 요소 데이터 업로드 가이드

## 개요
마을 안전 사각지대 예측 지도에서 **대량의 위험 요소 데이터(싱크홀, 가로등 파손 등)를 한 번에 업로드**할 수 있습니다.

## 지원 파일 형식
- **CSV** (.csv)
- **Excel** (.xlsx, .xls)

## 필수 데이터 컬럼

### 1. 좌표 정보 (필수)
| 컬럼명 | 설명 | 범위 |
|--------|------|------|
| `latitude` 또는 `위도` | 위도 | 37.41 ~ 37.72 |
| `longitude` 또는 `경도` | 경도 | 126.76 ~ 127.18 |

### 2. 위험 유형 (필수)
| 컬럼명 | 설명 |
|--------|------|
| `type` 또는 `종류` | 다음 중 하나: `조명 부족`, `시야 차단`, `도로 파손`, `불법 주정차` |

### 3. 위험도 (필수)
| 컬럼명 | 설명 | 범위 |
|--------|------|------|
| `intensity` 또는 `위험도` | 위험 정도 | 1(약함) ~ 5(심각함) |

### 4. 설명 (선택)
| 컬럼명 | 설명 |
|--------|------|
| `description` 또는 `설명` | 상세 설명 (선택) |

## 업로드 방법

1. **앱 실행**
   ```bash
   python -m streamlit run app.py
   ```

2. **왼쪽 사이드 패널 접기 해제** (사이드바 확장)

3. **"📤 데이터 업로드" 섹션에서 파일 선택**
   - CSV 또는 Excel 파일 업로드

4. **검증 및 프리뷰**
   - 자동으로 데이터가 검증됨
   - 문제가 없으면 "✅ 업로드 완료" 메시지 표시
   - 미리보기에서 업로드된 데이터 확인

5. **지도에서 즉시 표시**
   - 업로드된 위험 요소들이 지도에 마커로 표시됨
   - 마커 클릭 시 세부 정보 확인 가능

## 검증 규칙

### 필수 사항
- [ ] 모든 필수 컬럼 존재
- [ ] 위도: 37.41 ~ 37.72 (서울 범위)
- [ ] 경도: 126.76 ~ 127.18 (서울 범위)
- [ ] 위험도: 1 ~ 5 (정수)
- [ ] 위험 유형: 정확히 4가지 중 하나

### 검증 실패 시
- 오류 메시지로 어느 부분이 잘못되었는지 표시
- 파일을 수정 후 다시 업로드

## 위험 유형 설명

| 아이콘 | 유형 | 설명 |
|--------|------|------|
| 💡 | 조명 부족 | 가로등 고장, 어두운 골목 등 |
| 🚧 | 시야 차단 | 담벼락, 공사 가림막 등 시야를 방해하는 물체 |
| 🕳️ | 도로 파손 | 포트홀, 보도블럭 손상, 맨홀 파손 등 |
| 🚗 | 불법 주정차 | 통학로, 보호구역의 불법 주정차 |

## 위험도 기준

| 레벨 | 설명 | 예시 |
|------|------|------|
| 1 | 약함 | 가로등 약간 어두움, 작은 포트홀 |
| 2 | 보통 | 가로등 1개 고장, 중간 크기 손상 |
| 3 | 중간 | 여러 가로등 고장, 보행 어려움 |
| 4 | 심각 | 심각한 조명 부족, 큰 포트홀 |
| 5 | 매우심각 | 통행 위험, 즉시 조치 필요 |

## 자주 묻는 질문 (FAQ)

### Q: 업로드할 수 있는 최대 데이터 개수는?
A: 특별한 제한은 없으나, 성능상 10,000개 이상은 권장하지 않습니다.

### Q: 기존 데이터를 지우고 새로 업로드할 수 있나요?
A: "🗑️ 업로드 데이터 초기화" 버튼으로 기존 데이터를 지운 후 새로 업로드할 수 있습니다.

### Q: 한글과 영문 컬럼명을 섞어서 사용 가능한가요?
A: 네, 시스템이 자동으로 인식합니다.
- 영문: `latitude`, `longitude`, `type`, `intensity`, `description`
- 한글: `위도`, `경도`, `종류`, `위험도`, `설명`

### Q: 위험 유형은 정확히 뭘로 입력해야 하나요?
A: 정확히 다음 4가지만 가능합니다:
- `조명 부족`
- `시야 차단`
- `도로 파손`
- `불법 주정차`

### Q: 좌표를 잘못 입력했어요. 수정할 수 있나요?
A: 파일을 수정 후 다시 업로드하거나, "🗑️ 업로드 데이터 초기화"로 초기화 후 다시 업로드할 수 있습니다.
